Member Management API

Welcome to the Member Management API demo page. This API allows you to manage member information securely.

Getting Started

To use this API, follow these steps:

  1. Access the API using the base URL: https://tharanitharan.pythonanywhere.com/
  2. Use an HTTP client like Postman or cURL to make requests.
  3. For protected endpoints, use the following credentials:
    • Username: admin
    • Password: password

Available Endpoints

  • GET /member - Get a list of all members.
  • GET /member/<member_id> - Get details about a specific member.
  • POST /member - Add a new member to the database.
  • PUT /member/<member_id> - Edit information of a member.
  • PATCH /member/<member_id> - Partially edit information of a member.
  • DELETE /member/<member_id> - Delete a member.

Example Request

Here's an example of how to retrieve a list of all members:

            
                GET /member
                Authorization: Basic YWRtaW46cGFzc3dvcmQ=

Remember to replace YWRtaW46cGFzc3dvcmQ= with the actual Base64-encoded credentials.
